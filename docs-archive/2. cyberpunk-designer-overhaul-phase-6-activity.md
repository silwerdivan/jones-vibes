# Phase 6: Onboarding Reset and Labor Sector Mobile Rebuild

## Summary
This phase removes the decorative faux-EULA flow and replaces it with a concise introduction that explains the actual game. It also repairs the remaining mobile UI regressions that made Labor Sector feel crowded, kept hustles below the fold, and weakened the HUD avatar presentation.

## Tasks

### 1. Replace onboarding with a real introduction
- [x] `src/ui/components/EulaModal.ts`
  - [x] Remove the multi-step Neural Sync flow, opponent setup, and optional protocol enhancements.
  - [x] Present a single-screen introduction that explains the loop: move, spend time, manage credits, manage sanity, and survive weekly pressure.
  - [x] End with one primary CTA that starts the run.
- [x] `src/data/eula.ts`
  - [x] Replace clause-driven onboarding copy with introduction copy and concise callouts.
- [x] `src/main.ts`
