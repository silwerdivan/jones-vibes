# Project: Cyberpunk Designer Overhaul - Phase 3

## Objective
Implement Phase 3 of the MDA framework: "The Hustle & Narrative Tension" while addressing current game-breaking glitches.

## Core Feature: The Gig-Board / Hustles
- Implement a new "Hustle" action in the Labor Sector (e.g. "Donate Blood-Plasma" or "Scrap Metal").
- Hustles provide immediate Credits but incur massive Energy/Sanity costs or run a high risk of negative consequences.

## Tacked-on Thematic Change: The Event Deck Expansion
- Add 3 new narrative tension events to `randomEvents.ts` (e.g., "Panic Attack on the Mag-Lev" or "Shady Fixer Courier Job") that check against Sanity and Energy.
