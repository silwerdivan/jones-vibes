# Project: Cyberpunk Designer Overhaul - Phase 4

## Objective
Repair the broken economy pressure and dead upgrade math so the "poverty trap -> ruthless empowerment" arc is enforced by the numbers instead of only implied by the fiction.

## Core Feature: Real Insolvency
- Make missed Burn Rate payments mechanically painful instead of cosmetic.
- Ensure unpaid weekly costs create debt, pressure, or escalating penalties that the player cannot ignore by sitting at 0 credits.

## Tacked-on Thematic Change: Hustle Rebalance
- Re-tune early hustles so they remain tempting emergency relief valves, but stop outperforming stable labor on pure efficiency.

## Lessons Learnt (Phase 4)
- **Compound Pressure Works:** Debt interest and the `SUBSCRIPTION_DEFAULT` condition create a meaningful downward spiral that makes "just sitting at 0 credits" impossible as a strategy.
- **Modifier Multipliers:** Correctly chaining `WAGE_MULTIPLIER` and `WORK_EFFICIENCY` allows for satisfying power progression (e.g., BIO_SYNC + Adrenaline Pump).
- **History-Based Cooldowns:** Event cooldowns are limited to the last 3 events, meaning a small deck will repeat events quickly. This is intended to keep the pressure high but might need adjustment if the deck grows.
- **Testing for Debt:** Tests that rely on end-turn transitions must now account for Burn Rate debt if the player starts with 0 credits.
